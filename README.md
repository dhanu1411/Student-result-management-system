# Student Result Management System

This is a simple Python project that stores student marks and generates results in Excel format.

## Features
- Calculates total marks
- Calculates average
- Assigns grades
- Saves data into Excel file

## Technologies Used
- Python
- openpyxl
- Excel

## How to Run

1. Install Python
2. Install openpyxl:
   pip install openpyxl

3. Run:
   python main.py
